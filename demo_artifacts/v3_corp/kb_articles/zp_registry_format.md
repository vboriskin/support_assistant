# Зарплатный реестр: формат и валидация

## Формат файла
- **XLSX** только (не XLS).
- **Версия шаблона v3.2** — с 2026-04-01. Старая v3.1 больше не принимается.
- Скачать шаблон: в кабинете → Зарплатные → «Скачать пустой реестр».

## Обязательные колонки
| Колонка | Тип | Пример |
|---|---|---|
| ТАБ_NO | string | T-001234 |
| ФИО | string | Иванов Иван Иванович |
| КАРТА | string | 4276XXXXXXXXXX12 (16 цифр, можно без пробелов) |
| СУММА | number | 75000.00 (рубли с копейками) |
| НАЗНАЧЕНИЕ | string | Аванс за апрель 2026 |

## Типичные ошибки

- **«Неверный формат»** — старый шаблон v3.1. Скачайте свежий.
- **«Карта не найдена»** — сотрудник заблокировал карту или перевыпустил. Свяжитесь с HR, обновите данные.
- **«Сумма дробная не та»** — точка-разделитель, не запятая. 75000.00, а не 75 000,00.
- **«Превышен лимит ФОТ на день»** — для крупных компаний нужна **подпись двух уполномоченных лиц**.

## Корректировки

Если один сотрудник «выпал» из выплаты — подать **корректировочный реестр** (Зарплатные → Корректировка → Создать). В нём только тот, кого корректируете.
