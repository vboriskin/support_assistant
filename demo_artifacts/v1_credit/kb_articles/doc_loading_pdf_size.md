# Загрузка PDF — ограничения по размеру

## Симптом
При загрузке PDF >10 МБ страница падает с ошибкой 500 или зависает на 95%. Файл не появляется в карточке клиента.

## Причина
Ограничение на размер вложения — 10 МБ. На скан паспорта с высоким DPI (≥600) это не всегда хватает. Лимит выставлен на стороне бэкенда (nginx + FastAPI body limit).

## Решение
1. Открыть PDF в Acrobat → «Уменьшить размер файла» → выбрать «Совместимость с Acrobat 7.0 и более поздними версиями».
2. Целевой размер — до 5 МБ. Этого хватит для OCR.
3. Загрузить заново через «Документы → Загрузить».
4. Если PDF — фотография паспорта, попросить клиента переснять при равномерном освещении сверху, без бликов — итоговый файл будет меньше.

## Если не помогло
- Эскалировать на **support_l2_docs**.
- Приложить external_id заявки и пример файла (без PII).
- Если повторяется массово — открыть инцидент: возможно, забит storage.
