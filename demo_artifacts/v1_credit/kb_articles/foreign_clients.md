# Иностранные клиенты

## Кому доступен кредит
- Граждане РФ.
- Иностранцы с **видом на жительство** (ВНЖ).
- Иностранцы с **разрешением на временное проживание** (РВП).
- Беженцы — по отдельной программе через L2.

## НЕ доступен
- Туристы и иностранцы без РВП/ВНЖ.
- Иностранцы со студенческой/рабочей визой без РВП.

## Документы

| Документ | Гражданин РФ | Иностранец ВНЖ | Иностранец РВП |
|---|---|---|---|
| Паспорт РФ | ✓ | — | — |
| Паспорт страны | — | ✓ + перевод | ✓ + перевод |
| ВНЖ/РВП | — | ✓ | ✓ |
| СНИЛС | ✓ обяз | желательно | желательно |
| ИНН | ✓ | ✓ | ✓ |
| Миграционная карта | — | ✓ | ✓ |
| Подтверждение дохода в РФ | ✓ | ✓ | ✓ (минимум 12 мес) |

## Перевод документов
Только нотариальный. Принимается с печатью переводчика + апостиль.

## Двуязычный ДКП
Не предоставляется. Клиент может заказать нотариальный перевод за свой счёт после подписания.

## Эскалация
**support_l2_legal** — все вопросы про иностранцев. Не L1.
