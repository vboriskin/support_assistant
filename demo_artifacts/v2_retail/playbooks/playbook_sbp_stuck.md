# Шаблон ответа: перевод по СБП висит

## Когда применять
Клиент перевёл по СБП, через >15 минут получатель не видит деньги, либо у клиента в приложении статус «В обработке».

## Шаблон
Здравствуйте!

По вашему переводу:

- Если статус «В обработке» — деньги уйдут получателю автоматически в течение 2 часов либо вернутся вам на счёт. Это происходит при системных задержках на стороне СБП.
- Если статус «Проведён» — деньги уже у получателя; уточните у него, в какой банк вы переводили (иногда получатель ждёт зачисление в другом банке).

Чего точно НЕ нужно делать:
- Не отправлять повторно тот же перевод — это создаст дубль.
- Не оформлять «спор по карте» — для СБП этот механизм не работает.

Если через 2 часа ситуация не изменилась — напишите нам, и я подключу старшую линию.

С уважением,
Поддержка
