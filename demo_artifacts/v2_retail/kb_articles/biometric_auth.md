# Биометрия (Face ID / Touch ID) — типовые проблемы

## После обновления ОС не работает

Самая частая причина. iOS / Android после крупного обновления сбрасывают разрешения на биометрию.

**iOS**: Настройки → Банк → Face ID → включить.
**Android**: Настройки → Приложения → Банк → Разрешения → Биометрия → включить.

После этого войти в приложение **один раз по PIN**, затем биометрия снова заработает.

## Биометрия отключена принудительно

- 3 неуспешных попытки → отключение биометрии. Включается заново после успешного входа по PIN.
- Смена биометрии на устройстве (добавление нового лица в Face ID) — приложение требует подтверждение через PIN.

## Не настраивается биометрия в приложении

- Биометрия не настроена в самой ОС → сначала настроить в Настройках устройства.
- Устройство **рутовано / jailbreak** — биометрия принудительно отключается из соображений безопасности.
