# Push-уведомления о транзакциях

## Не приходят пуши

Проверять в порядке возрастания цены:

1. **Настройки приложения** → Уведомления → Все включены.
2. **Системные настройки** → ОС → Уведомления → Банк → разрешены (Banners + Sounds).
3. **Режим «Не беспокоить»** / Focus Mode на устройстве — может фильтровать.
4. **Battery Saver на Android** — отключает фоновые pushes для приложения. Открыть Настройки → Аккумулятор → Банк → «Не оптимизировать».
5. Старая версия приложения — обновить через store.

## Задержка пушей

- iOS — обычно <5 сек.
- Android — может быть до 5 минут при энергосбережении. См. Battery Saver выше.

## Пуш пришёл, но не открывается

- Закрыть и переоткрыть приложение.
- Если пуш про «крупную операцию» (>30 000 ₽) и клиент её не совершал — НЕ открывать ссылку из пуша, а звонить в 900.

## Эскалация
**support_l1** — обычные случаи (настройки).
**platform_team** — если у нескольких клиентов сразу.
